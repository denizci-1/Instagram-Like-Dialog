# MyLog
# Use Instagram-like dialog.

# HOW TO IMPORT

Step 1. Add the JitPack repository to your build file

```
allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
```
Step 2. Add the dependency
```
dependencies {
            implementation 'com.github.iamkurtgoz:Instagram-Like-Dialog:1.0.1'
}
```
